# Folklore

League of Legends chat library for iOS.

## Installation

Add the following to your [`Podfile`](http://docs.cocoapods.org/podfile.html)
and run `pod install`

```
pod 'Folklore'
```

Don't forget to `#import <Folklore/Folklore.h>`.
